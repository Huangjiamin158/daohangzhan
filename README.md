# 开发者工具导航站

这是一个简单的单页面开发者工具导航网站，专注于为独立开发者提供出海相关的工具和资源导航。

## 功能特点

- 响应式设计，适配桌面和移动设备
- 简洁的卡片式布局
- 搜索功能，方便快速查找资源
- 分类清晰的六大模块，对应不同需求

## 技术栈

- HTML5
- CSS3
- 原生 JavaScript

## 使用方法

直接打开 `index.html` 文件即可使用。

## 目录结构

```
.
├── README.md
├── index.html
├── styles.css
└── script.js
```

## 维护说明

如需添加新的工具或资源，请直接编辑 `index.html` 文件中相应的分类部分。
